# Sample Inputs

These are example inputs that work well with the predictor.

## Text Snippet

```text
标题一行
正文几行...
https://example.com/some-article
```

## Telegram Public Post Link

```text
https://t.me/<channel>/<id>
```

Tip: using a public `t.me` link lets the predictor extract publish time for weekday/hour features.
